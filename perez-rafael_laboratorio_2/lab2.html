Laboratorio #2 - Problemas básicos con Javascript

1. ¿Qué es Javascript y en qué se asemeja a Java?

JavaScript es un lenguaje de programación orientada a objetos, basados en prototipos, imperativo, debilmente tipado y dinámico. Se asemeja con java en cuanto ambos proceden del lenguaje de programación C, por lo tanto tienen la misma estrutura y mismo nombre.
